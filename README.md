This repository is built to provide an advanced constraint-based timetable generator designed specifically for schools following KVS timetable guidelines.

_**The system intelligently handles:**_
- Teacher workload limitations
- Class availability constraints
- Lab period allocation for Classes 11 & 12
- Mathematics period distribution for: _Bio-Maths students_, _Computer Science–Maths students_, _Commerce with Maths students_
- Structured scheduling with fully enforced constraints
- Balanced and conflict-free timetable construction
- Every timetable generated is logically structured, optimized, and constraint-aware.



_**🚀 Transforming School Timetable Management**_

- Built in alignment with the visions of Viksit Bharat and Digital India, this application aims to modernize school systems — transitioning from outdated manual scheduling methods to efficient, technology-driven planning.
- The goal is simple:
    " Make schools smarter, more organized, and future-ready."


_**🏫 Current Features**_

- Generates timetables for Classes 6 to 12
- Three sections per class (in app and can be changed)
- Teacher-wise timetable generation
- Class-wise timetable generation
- Day-wise timetable generation
- Fully offline functionality
**(Some feature updates may require internet access, like updating to a newer version)**

_**🐧 Linux Installation Manual**_

**1️⃣ System Requirements**

- 64-bit Linux OS
(Ubuntu, Kali, Debian, Fedora, Arch, Pop!_OS recommended)
- Latest Python version


**2️⃣ Install Required Dependencies**

- Open Terminal:
 (or just press ctrl + alt + t)

**For Ubuntu / Kali / Debian:**

_____________________________________________________________________
sudo apt update                                                      
sudo apt install -y build-essential libssl-dev zlib1g-dev \       
libncurses5-dev libncursesw5-dev libreadline-dev libsqlite3-dev \
libgdbm-dev libdb5.3-dev libbz2-dev libexpat1-dev liblzma-dev \
tk-dev libffi-dev wget
_____________________________________________________________________



**For Fedora:**

_______________________________________________________________
sudo dnf groupinstall "Development Tools"
sudo dnf install openssl-devel bzip2-devel libffi-devel wget
_______________________________________________________________



**For Arch:**
____________________________________________________
sudo pacman -S base-devel openssl zlib xz tk wget
____________________________________________________



**3️⃣ Download the Application**

- Go to the Releases section of the repository.
- Download the file named "main".
- It is strongly recommended to download it into a separate folder to avoid clutter during updates.


**4️⃣ Make the File Executable**

- Open the folder where the file is saved.
- Right-click the file → Click Properties.
- Enable "Executable as Program".
- Double-click the main file to run it.

- On first launch:
**1. The app will prompt for updates.**
**2. Click Yes and allow it to complete installation.**

**5️⃣ Using the App**

- After setup, you can:
1. Modify teacher names
2. Adjust class availability
3. Assign subjects
4. Generate the timetable in the format for:
(i) Class-wise timetable
(ii)Teacher-wise timetable
(iii)Day-wise timetable
- Timetable management becomes faster, structured, and far more efficient.
